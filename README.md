# cicd-project

## 🚀 CI/CD Pipeline Project

This project demonstrates a complete CI/CD pipeline using AWS services.

## 🛠️ Tools Used
- AWS CodePipeline  
- AWS CodeBuild  
- AWS CodeDeploy  
- Amazon EC2  
- GitHub  

## 📌 Workflow
GitHub → CodePipeline → CodeBuild → CodeDeploy → EC2
